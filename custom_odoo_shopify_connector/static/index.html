How the automation works

The connector listens to Shopify order webhooks and processes them in a background queue. Confirmed Odoo sales then drive delivery and accounting.

  • 1
    Customer places an order in Shopify
    Shopify sends an orders/create webhook to your Odoo endpoint /shopify/webhook/order.
  • 2
    Order pushed into the Odoo queue
    The webhook handler validates the signature and stores the payload as a shopify.order.queue record marked as Pending.
  • 3
    Background job creates the Sale Order
    The cron job “Process Shopify Order Queue” runs every minute, calling the order service to create an sale.order, add lines and automatically confirm it (generating the delivery order).
  • 4
    Warehouse and accounting take over
    Odoo stock users process the outgoing delivery. Depending on your Odoo settings you can then create and post invoices and register payments.

How to use this module

Follow these steps after installing the module in your Odoo database.

  • A
    Create a Shopify Store in Odoo
    Go to Sales ▸ Configuration ▸ Shopify Stores, create a record and fill in: Shop URL, Access Token, optional API key/secret and company.
  • B
    Configure the webhook in Shopify
    In your Shopify admin create a webhook for Orders → orders/create pointing to https://YOUR_ODOO_DOMAIN/shopify/webhook/order and copy the shared secret into the Odoo store’s Webhook Secret field.
  • C
    Verify connection
    From the Shopify Store form click Test Connection to ensure the API credentials are valid.
  • D
    Monitor the order queue
    Use the Shopify ▸ Order Queue menu to see pending, processing, done and failed queue entries for troubleshooting.
